# 视频号标题与封面策略

视频号不是简单复用 B站/抖音策略。

## 流量特征

| 特征 | 对内容的影响 |
| --- | --- |
| 社交推荐为主 | 内容要让人愿意转发：民族自豪、知识获得感、大国重器 |
| 小图卡片预览 | 封面文字比 B站更大，居中，少字 |
| 25-55 岁主力用户 | 科技自立、产业升级、制造业、能源、军工更适配 |
| 不靠强算法推流 | 标题不必堆热点词，要有情绪钩子 |
| 描述首行可见 | 前 20 字决定点击 |

## 标题规则

- 建议 <= 15 字
- 优先画面感和数据冲击
- 避免平台味太重的标题党
- 每日新中国禁用 emoji

## 封面规则

- 竖版 `1080x1440`
- 大号白字居中
- 半透明黑底保证小图可读
- 优先使用当天最有传播力的 GPT 图

## 2026-07-08 实例

主线：人形机器人产量今年有望突破十万台。

采用：

```text
标题：人形机器人将进厂打工
封面：人形机器人生产线图 + 大字居中
```
